## 词法分析

把程序中的每个不可分割的最小单位，变成方便编译器处理的统一的格式(称作token)，扔掉多余的空格，注释什么的

统一的格式{符号类型，符号具体内容}

例子

```c++
int main(){
    printf("hello,world");  //这是个注释
}
```

词法分析的结果例子

```javascript
{关键字int，}
{标识符，"main"}
{左小括号，}
{右小括号，}
{左大括号，}
{标识符，"printf"}
{左小括号，}
{字符串常量，"hello,world"}
{右小括号，}
{分号，}
{右大括号，}
{文件结束EOF，}
```

这样后面的分析中就可以用一个统一的对象集合来处理语句了，而不用考虑很多字符串处理中的复杂问题
